lancsgrid_vac
=========

An Ansible role that installs and configures vac on SL/RHEL/CentOS 6.x .

Requirements
------------

So far only tested on an SL6 machine.

Role Variables
--------------

None.

Dependencies
------------

None.

Example Playbook
----------------

    - hosts: servers
      roles:
         - { role: username.vac}
